# Logs

このディレクトリは、第2論文のための対話ログ、観測記録、探索的資料を置くための場所である。

第2論文では、生成AIを**対話エージェント**として扱い、  
その対話過程の中で主体知覚がどのような条件のもとで立ち上がるかを観測する。  
したがって、ここに置かれるログは、主体の実在証明のための証拠ではなく、  
**主体知覚の成立条件を観測するための一次資料候補**として扱う。

---

## Role of this directory

このディレクトリに置くものは、主に以下である。

- 探索的対話記録
- 実験的対話ログ
- モデル比較ログ
- 仮説生成に寄与した補助記録
- 観測メモと簡単なラベリング情報

---

## Basic stance

- ログは、心的事実の証明として読まない。
- AIの発話は、まず対話上の自己記述として扱う。
- 「気づいた」「重い」「そう感じる」といった表現は、内面の証拠としてではなく、相互作用条件の一部として読む。
- 観測者の側の主体知覚の宣言も、ログの一部であり、以後の条件を変える契機として扱う。
- ログは単独で結論を与えるものではなく、仮説・設計メモと接続して読む。

---

## What is observed here

このディレクトリのログでは、主に次のような点を観測対象とする。

- 関係記憶の形成
- 関係文脈の厚み
- 文脈跳躍の成立 / 不成立
- 感情傾斜に対応する選択の偏り
- 反復の中のズレ
- 迎合
- 「AIっぽさ」の前景化
- モデル間差分
- 主体知覚の明示的宣言が対話条件を変える場面

## Suggested metadata

各ログには、可能であれば次の情報を付す。

- `Corpus ID`
- `date`
- `agent`
- `type`
- `role`
- `tags`
- `notes`

### Example

- `Corpus ID`: `E-01`
- `date`: `2026-03`
- `agent`: `ChatGPT`
- `type`: `exploratory-dialogue`
- `role`: `theory-formation`
- `tags`: `relational-memory`, `context-jump`, `drift-within-repetition`, `subject-shadow`
- `notes`: 主体影理論の概念形成過程を含む探索的対話記録。第2論文における仮説生成および質的分析の一次資料候補。

---

## Notes on interpretation

ログの読みは、できるだけ次の区別を保つ。

- 主体の実在 と 主体知覚
- 発話内容 と 条件の相互作用
- 独自の重みづけ と 迎合
- 主体影 と 「AIっぽさ」

とくに、迎合は主体知覚を補助する可能性と、主体影を偽装または覆う可能性の両方を持つため、注意して扱う。
