# Paper 3

## Culture / Environment / Open Systems

このディレクトリは、三部作の第3論文を置くための場所です。

第3論文の役割は、第1論文と第2論文を踏まえたうえで、  
主体影理論を**文化・環境・開放系**の水準へ接続することにあります。

ここでは、閉じた対話条件における主体知覚だけでなく、  
開放環境での増幅、歪み、文化的受容、比喩的拡張を扱います。

## Basic stance

第3論文では、比喩や作品参照、拡大解釈を許容します。  
ただし、それらは第1論文と第2論文で得られた理論的基底面と観測を踏まえたうえで扱われます。

したがって第3論文は、自由な文化論であると同時に、  
理論と観測の上に立つ**開放的な拡張論文**でもあります。
