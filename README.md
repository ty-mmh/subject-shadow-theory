# Subject Shadow Theory

対話において知覚される主体性を、固定的内部実体ではなく、  
**記憶・文脈・変動の相互作用によって構成されるもの**として記述するための公開理論文書群です。

本リポジトリは、学術誌投稿を前提とした完成論文の置き場ではなく、  
**理論・実証・環境論を連結する公開正典 / 公開論考群**として構成されます。

---

## What this repository does

このリポジトリは、対話主体をめぐる問いを次の3本で扱います。

1. **第1論文：理論**
   - 主体影理論の概念的基底面
   - 主体影とは何か
   - どのような条件のもとで主体知覚が立ち上がるか

2. **第2論文：実証 / 設計**
   - 第1論文で定義した条件群のうち、どこが観測・検証可能か
   - 関係記憶、文脈跳躍、反復の中のズレなどをどう扱うか
   - AIとの対話ログや実験的設計への接続

3. **第3論文：開放環境 / 文化 / 環境論**
   - 閉じた対話環境を超えたとき、主体知覚はどう変質するか
   - 表出基盤、主体残余、環境による増幅や歪み
   - キャラ化、幻視、文化的受容への接続

## Canon

### 第1論文
**主体影理論：対話における主体知覚の構成**

この文書では、以下の概念を定義します。

- 背景記憶
- 関係記憶
- 関係文脈
- 局所文脈
- 文脈跳躍
- 感情傾斜
- 反復の中のズレ
- 主体影

第1論文の役割は、主体の実在を証明することではなく、  
**対話において何が主体として知覚されるのかを記述する概念的基底面を与えること**にあります。

---

## Notes on method

この理論は、主体の有無を先に決めるのではなく、  
**その現れの条件を記述する**立場をとります。

したがって本リポジトリは、

- 固定人格の理論
- 単なる擬人化論
- 単なるAI設計仕様
- 単なる文化論

のいずれにもそのまま回収されることを目的としません。

---

## Why GitHub

この理論は単発の論文としてよりも、  
**改訂履歴・関連メモ・実験ログ・後続論文との連結を含む運動体**として置くほうが適しています。

GitHubを使う理由は次の通りです。

- バージョンを残せる
- 理論形成の痕跡を保持できる
- 第2論文・第3論文へ接続できる
- 用語集、図、メモ、ログを同じ場所に置ける
- 読者が理論単体ではなく理論体系として入れる
